# User settings

User's settings are available trough e.g. http://bacula-web.domain.com/user

![Bacula-Web user settings](../assets/bacula-web-user-settings.jpg)

:::info
As of the latest 9.x Bacula-Web release, user settings modifications are limited to the user's password reset only.
:::

## Password reset

Each user can reset their password from the user settings page by using the **Password management** form.

![Bacula-Web user password reset](../assets/bacula-web-user-password-reset.png)
