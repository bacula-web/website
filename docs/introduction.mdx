# Introduction

Welcome to the official documentation of Bacula-Web, the open source reporting and monitoring tool for [Bacula](https://bacula.org).

You can access the documentation for each topic from the table of content in the sidebar, you can also use the search bar in the top-right corner.

If you want to know more about the project, we recommend that you read the [About page](about/index.md).

For new installation, please read the [installation](install/getting-started.md) page.

To upgrade Bacula-Web, please see the [Upgrade](install/upgrade.md) page.

## Get involved and contribute

Bacula-Web is an open source project developed and maintained by a community of volunteers.

If you want to get involved in improving the current documentation, fix a bug in the code, or add a new feature,
then read the [Contribution guide](contribute/index.md).

## How to get Help

At any time, if you have a question or any feedback, you can

* contact me by mail at hello@bacula-web.org
* use the [GitHub discussion](https://github.com/bacula-web/bacula-web/discussions)
* send an email to the Bacula Users mailing list at bacula-users@lists.sourceforge.net

If you want to submit an issue or a feature request, there's also a [bug tracker](https://github.com/bacula-web/bacula-web/issues) available on the GitHub project.

The [Get Help](gethelp/index.md) page should also contain some information to help you.
