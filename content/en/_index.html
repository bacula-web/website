Getting started

{{< card-code header="" lang="C" >}} # Create using composer $ composer create-project bacula-web/bacula-web # Setup the application $ cd bacula-web $ sudo -u www-data php bwc setupauth # Check you meet all requirement $ sudo -u www-data php bwc check # Go to the server url e.g: http://bacula-web.domain.com That's it ! {{< /card-code >}} {{< /blocks/section >}}
