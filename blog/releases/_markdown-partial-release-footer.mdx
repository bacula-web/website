:::info[Change on Web server public folder]
Kind reminder, since [release v8.6.0](https://github.com/bacula-web/bacula-web/releases/tag/v8.6.0), Apache, Nginx or Lighttpd document root folder muse be set as below

`WEB_SERVER_ROOT/bacula-web/public`

> In the example above, you can replace WEB_SERVER_ROOT by /var/www/html (adapt to your environment
:::

## How to install / upgrade ?

The installation or upgrade instructions are available in the [official documentation](https://www.bacula-web.org/docs).

## Docker images

Latest Bacula-Web Docker images (`linux/amd64` and `linux/arm64`) are available at [Docker Hub](https://hub.docker.com/r/baculaweb/bacula-web)

:::tip
Docker images are available only since [release v8.7.0](https://github.com/bacula-web/bacula-web/releases/tag/v8.7.0).
:::

## How to get help ?

They are several options to get help

* use [GitHub discussions](https://github.com/bacula-web/bacula-web/discussions) to ask a question or give a feedback
* read the [FAQ](/docs/gethelp/faq)

## Found a bug, or need a new feature ?

You can submit a new bug report or feature request on the [GitHub project](https://github.com/bacula-web/bacula-web/issues) (please read the [bug report guideline](/docs/gethelp/support))

## How to contribute ?

Bacula-Web is an open source project, mostly maintained on my spare time with the great help from community users.

If you're willing to contribute to the project, have a look at the [contribution guide](https://www.bacula-web.org/docs/contribute/)

## Security notice

:::warning
Some versions of Bacula-Web have CVEs due to identified security vulnerabilities, detailed in the [security page](/docs/about/security).

I **strongly encourage** anyone using Bacula-Web to continue using the latest stable releases, which include bug fixes and security patches.
:::
